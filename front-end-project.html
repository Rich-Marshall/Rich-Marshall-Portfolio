Front End Project

An analysis of an HTML, CSS and Javascript project. The project is a guide to the ski resort Val d'Isere. This analysis shows use of HTML, CSS and Javascript. It also showcases version control through GIT

Overview

This project was undertaken as part of a front-end web development module at university. The site is purely HTML, CSS and JavaScript. And uses GIT version control.

Aims

The project aimed to use APIs and HTML CSS and JavaScript to produce a site for a chosen topic. I chose to develop an information site for the ski resort, Val d’Isere. The site needed to be comprised of 4 individual HTML pages with JavaScript and API usage on some or all of those pages.

Tools

During development I made extensive use of GIT and GitHub and during this time I learned version control which has proved very helpful during most of my projects since. This project also focused heavily on mobile-first design and each page is fully responsive to browser size.


Development

One aspect of this site had to include JavaScript and so I developed a draggable, zoomable piste map for people to use when in the hotel instead of having to use the paper maps supplied by the resort.

The landing page displays a real time clock for the time in resort and acts as a launch point for the rest of the site.

The other pages are simpler and provide information about the resort and a full bar guide for all those party goers looking for some après! The final page pulls data from the snow forecast API and displays snow reports and weather reports for the resort that day. It updates every 24 hrs.

Outcomes

The site was very successful and achieved all of its aims. I learned a lot about mobile first design and API usage. I was able to gain a good understanding of version control. Furthermore, the site functions well and has been used by the university SNOW society who travelled to Val d’Isere the year this project was completed.
